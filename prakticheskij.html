Как работать с практическими заданиями

  1. Фундамент (1-4 балла): решение по образцу — для всех учащихся.
  2. Стены (5-7 баллов): типовая профессиональная задача — для уверенных.
  3. Кровля (8-10 баллов): нестандартная задача — для сильных.
  4. После выполнения: сверьтесь с ответами, разберите ошибки.
